Collect Contributions💰

As a team leader organizing an event, I want to collect contributions from a group, so I can make payments without using my own money.

Automatic Splits⚖️

As a roommate sharing rent and utilities, I want to set up automatic recurring expense splits, so we can fairly distribute monthly bills without constant manual calculations.

Flexible Payments💳

As a user with different financial accounts, I want to choose how I settle my balance (e.g., PayPal, Venmo, bank transfer), so I can pay in the way that’s most convenient for me.

Travel Expense Splits✈️

As a frequent traveler with friends, I want to easily split expenses for accommodations, food, and activities, so I don’t have to manually track and calculate who owes what.

Receipt Splitting🍽️

As a restaurant-goer, I want to scan a receipt and assign items to individuals, so that each person only pays for what they ordered, including tax and tip.

Expense Management📊

As a finance-conscious user, I want to view and export my past expenses in a spreadsheet, so I can keep track of my spending for budgeting or tax purposes.
